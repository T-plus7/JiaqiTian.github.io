# 个人主页（基于模板）

本仓库包含一个简单的静态个人主页模板，参考自示例站点，已包含基本布局、样式与交互逻辑。

创建的文件：
- [index.html](index.html)
- [css/styles.css](css/styles.css)
- [js/scripts.js](js/scripts.js)
- [assets/README.md](assets/README.md)

本地预览：在仓库根目录打开 `index.html`（或使用 VS Code Live Server）。

部署到 GitHub Pages：将此仓库推到你的 GitHub 仓库，然后在仓库设置的 Pages 中启用 `main` 分支的根目录作为发布源。
