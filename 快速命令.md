# Jekyll 服务器快速命令

## 启动服务器
```powershell
bundle exec jekyll serve -l -H localhost
```

## 重启服务器（修改 _config.yml 后）
1. 先停止当前服务器：在运行服务器的窗口按 `Ctrl+C`
2. 然后重新启动：
```powershell
bundle exec jekyll serve -l -H localhost
```

## 使用脚本（如果执行策略允许）

### 启动服务器
```powershell
.\start.ps1
```

### 重启服务器
```powershell
.\restart.ps1
```

## 如果脚本无法执行

### 方法 1：设置执行策略（推荐）
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
然后就可以正常使用 `.\start.ps1` 和 `.\restart.ps1` 了

### 方法 2：使用完整命令
直接复制粘贴以下命令到 PowerShell：

**启动服务器：**
```powershell
bundle exec jekyll serve -l -H localhost
```

**重启服务器：**
```powershell
Get-Process | Where-Object {$_.ProcessName -eq "ruby"} | Stop-Process -Force -ErrorAction SilentlyContinue; Start-Sleep -Seconds 2; bundle exec jekyll serve -l -H localhost
```
